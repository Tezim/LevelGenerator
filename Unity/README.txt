Use unity srcipts by attaching them to GameObjects. 
Tiles can be assigned arbitrarily.
For reference use Unity_demo project assets.
# Create empty unity 2D core project 
  Import Assets folder and assign objects in inspector
  Hit run and save generated tilemap by drag-and-drop into prefabs folder 
  For custom tiles -> tile size in demo project is 100x100px (Match size of tile to tilemap grid)
NOTE imput file with map has to be in specified format generated by LevelGenerator app
